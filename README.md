## UX Design – The 5 Planes

### 1. Strategy (Project Goals)

**Business Goal:** The key business goal is **to generate new student bookings**.  
**Main CTA:** “Book Now” button leads to the booking form to convert visitors into customers.

**User Goals:**

- New learners and returning drivers need clear information on services, pricing and testimonials.
- Drivers wanting confidence coaching or Pass Plus.
- People interested in manual, automatic, EV or towing lessons.
- Local residents looking for flexible, affordable training.
- Users need clear information, social proof (testimonials), and a simple way to request lessons on any device.

### 2. Scope (Features)

**Must Have (MVP)**

- User friendly navigation and responsive design
- Hero section with Call To Action
- Services and Pricing cards
- Fully functional Booking form on booking.html
- Success page after form submission
- Company Contact details are displayed clearly in the Footer

**Should Have**

- Testimonials - modal clickable cards
- Social media links in Footer
- Newsletter sign up

**Could Have**

- Calendar integration for lesson availability
- Event location map integration

### 3. Structure (Information Architecture)

- Consistent fixed navbar
- Home page: Hero - Services - Testimonials
- Dedicated booking page with form
- Success page
- Footer with contact and social links on every page

### 4. Skeleton (Wireframes)

The wireframes show:

- Fixed navbar
- Full-width hero
- Services as a column card grid
- Testimonials as clickable cards that open modals
- Booking form layout

## User Stories

**Must Have**

1. **As a site visitor**, I want a responsive navigation menu so that I can easily move between pages on any device type.
2. **As a prospective student**, I want a prominent hero section with a strong Call To Action so that I am immediately encouraged to book lessons.
3. **As a prospective student**, I want to see Services and Pricing cards so that I can choose the right product.
4. **As a prospective student**, I want a booking form so that I can submit my details and preferences easily.
5. **As a prospective student**, I want a clear Success page after submitting the Booking form so that I know my request was sent.
6. **As a site visitor**, I want to see the company’s Contact details displayed clearly in the footer so that I can easily find the address, phone number and email and feel confident contacting Safe Drive UK.

**Should Have**

1. **As a prospective student**, I want to read student testimonials that open in modals so that I can see real feedback without leaving the page.
2. **As a site visitor**, I want social media links in the Footer so that I can follow Safe Drive UK on X, Facebook and Instgram and contact the business.
3. **As a site visitor**, I want to sign up for the newsletter so that I can receive special offers and theory-test group dates.

**Could Have**

1. **As a prospective student**, I want to see lesson availability in a calendar so that I can choose convenient dates and times.
2. **As a prospective student**, I want to see the Safe Drive UK base and event locations on a map so that I can plan my journey.
